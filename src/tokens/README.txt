Aqui ficam seus arquivos de segurança fornecidos pela Google Cloud Platform.

**Não modifique esses arquivos ao menos se souber o que está fazendo, caso contrário poderá não funcionar mais essa aplicação**

Caso trabalhe com mais de um Classroom/Gmail, pode ser salvo seus tokens em algum lugar de sua responsabilidade e refazer uma nova 
configuração com outra credencial, e quando quiser voltar ao anterior basta copiar os dois arquivos anteriores de volta para esta pasta.

credentials.json  = é seu arquivo de credencial, essencial para criar seu token e operar a API.
Tokens.json = é seu arquivo de Token, contendo todos os dados de segurança para operar a API.

Ambos sempre devem estar junto nesta pasta para o funcionamento correto desta aplicação.
